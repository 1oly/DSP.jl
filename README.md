DSP.jl provides a number of common DSP routines in Julia.  So far, the following functions are
implemented:

### Periodogram estimation
* arraysplit
* periodogram
* welch_pgram
* bartlett_pgram

### Window functions
* rect
* hanning
* hamming
* tukey
* cosine
* lanczos
* triang
* bartlett
* gaussian
* bartlett_hann
* blackman
* kaiser

### Filter functions
* analogfilter
* digitalFilter

### Filter response types
* Lowpass
* Highpass
* Bandpass
* Bandstop

### Filter design methods
* Butterworth
